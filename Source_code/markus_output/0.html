2·河內人，天會十四年參與修北村湯王廟。
3·孟津人，進義校尉，大定四年任河南路同知兼知陜州。
4·正大三年守魯山縣尉，修立臺碑。
5·燕山人，明昌二年任萬泉縣令。

7·安邑人，進士。
8·濟寧人，約卒於明昌年間。
9·翼城人，大定間進士。
10·垣曲人，字湛然。幼聰明，通經術，年十六，為洪慶觀道士，賜以紫衣通玄號。年五十八，一日升几而坐，召徒作歌，歌竟飛升，其徒奉遺衣葬於觀北。
11·字藏用，大興府宛平人。皇統二年進士。曾任武清縣丞、磁州軍事判官、和川令、北京推官、大理司直，以丁憂去職。起復後歷任武定軍節度副使、大理丞、刑部郎中、祁州刺史、同知西京留守事。卒于陝西西路轉運使。
12·承安中任蒲城主簿。
13·道士，曾主修太清宮事。
14·許州大帥。蒙古軍陷許州，死之。

19·漢御史卜式第四十五代孫，大定之初任單州司候判官。
20·沁源縣人，曾師從閻寶山學道，大定年間度為道士。
21·良鄉人，清涼寺女道士，俗名房山，自幼年而悟，出家後，受恩戒。法名道堅，棄俗歸真四十餘年，泰和六年卒。

26·崔立事變後任兵部尚書，元帥左監軍。
